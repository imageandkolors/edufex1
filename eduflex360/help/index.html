Frequently Asked Questions
How do I reset a user's password?

To reset a user's password, navigate to the "Users" section in the admin panel. Find the user in the list, click the "Edit" button, and then select "Reset Password". An email with a password reset link will be sent to the user's registered email address.

What are the system requirements for installing EduFlex360?

EduFlex360 can be installed on various platforms:

  • VPS: Minimum 2GB RAM, 20GB SSD storage, Ubuntu 20.04 LTS or CentOS 8
  • cPanel: cPanel/WHM with PHP 8.1+, MySQL 5.7+, and sufficient disk space
  • Replit: 1GB RAM minimum, Node.js environment (for development/testing)

For detailed installation instructions, please refer to the Installation Guide.

How do I import student data in bulk?

To import student data in bulk, go to the "Users" section and click "Bulk Import". Download the CSV template, fill in the student details, and then upload the completed file. The system will validate the data and create user accounts for each student. You can also assign them to classes and sections during the import process.

Can I customize the grading system for my school?

Yes, EduFlex360 supports multiple grading systems. You can configure percentage-based, letter grade, GPA, and skill-based grading systems. Navigate to "Grading Systems" in the admin panel to create or modify grading scales. You can also assign different grading systems to different classes or subjects as needed.

How do I set up CBT exams?

To set up a CBT exam, go to the "Exams & Results" section and click "Create New Exam". Select "CBT" as the exam type. You can then import questions from a PDF or DOCX file, set exam parameters (duration, number of questions, randomization), and schedule the exam. Students will receive an access code to take the exam through the CBT portal.
